# HelpDesk3 - English

This application which I desinged at 10 years ago, and it was used in a company to manage the helpdesk tickets. 
The framwork was built using QT, C++ and MySQL, and it was designed to be simple and easy to use. the function is
not fully implemented, but it is still a good example of how to use QT and C++ to create a simple application.

Maybe I will update this project in the future, but for now, it is just a simple application that I created for share.

Thanks for reading this, and I hope you find it useful. If you have any questions or suggestions, please feel free to contact me.
